1.  MVC1    VS      MVC2
1) 구정방식 ★(면접문제)
- MVC1 : JSP가  MODEL, VIEW, CONTROLLER 역할을 모두 수행.(스파게티 코드)
- MVC2 :        MODEL(JAVA), VIEW(JSP), CONTROLLER(서블릿) 분리 

2) 유지보수성
- MVC1 : 낮음( JSP에 로직이 포함되어 재사용 어려움. ) - 간단한 웹
- MVC2 : 높음( MODEL, CONTROLLER의 재사용 ) - 대규모로 처리되어있는 웹 포트폴리오.
