문제
reducers 폴더 안에 있는 user.js 파일의 내용과 일치 하지 않아 테스트 파일을 실행했을때, 오류가 발생을 하게 된다.

느낀점
처음 배우는 내용들이라서 그런지 아직까지 어떤 식으로 진행이 되는지 전반적인 흐름 파악이 안되어 있는 상태이다.
흐름 파악만 되면 이러한 실수도 많이 줄어들 것 같다는 생각이 든다.
