# 트러블슈팅
--- 20251111
Caused by: java.sql.SQLSyntaxErrorException: ORA-00911: invalid character


Caused by: Error : 911, Position : 43, Sql = insert into milk values (milk_seq.nextval, #(mname), :1 , :2 ), OriginalSql = insert into milk values (milk_seq.nextval, #(mname), ?, ?), Error Msg = ORA-00911: invalid character
---

```
문제점
- config/db.properties 파일에서 db.url=jdbc:oracle:thin@localhost:1521:xe 에서 ':'빠져서 oracle을 가지고 올수 없게 되었다.

- #(manae)으로 입력하면 insert를 가지고 올 수 없게 된다.

해결방안
db.url=jdbc:oracle:thin@localhost:1521:xe 를 
db.url=jdbc:oracle:thin:@localhost:1521:xe 로 바꿔주면 제대로 oracle을 가지고 올 수 있게된다.

#(mname) 을 #{mname}으로 바꿔주면 정상적으로 insert구문이 작동을 하게 된다.


느낀점
oracle에서 spring으로 입력된 값들을 가지고 올때 driverclass, url, username, password 등을 잘 확인하고, 
oracle구문을 spring에서 정상적으로 작동시키기 위한 #{  }에 대한 이해를 위해 많은 연습이 필요함을 느꼈다. 
```
