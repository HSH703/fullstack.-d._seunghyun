#Track001 - git hub
1. git vs github
- git - 로컬에 파일의 변경이력 (내컴퓨터에 타임머신) 
- github - 클라우드올려서 협업 (친구들과 공유작업공간/친구들과 노는 놀이터) 

2. 기본명령어
   ` git init` 저장소 생성 (빈 상자 만들기)
   `git add . ` 변경된 파일 추가 ( 상자에 그림넣기)
   `git commit -m "설정" `  저장 ( 그림에 이름붙여저장)
